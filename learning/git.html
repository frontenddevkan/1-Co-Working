Просто введите git без всяких аргументов.

Git выведет свои параметры и наиболее общие подкоманды.

1/1 чтобы вывести полный список подкоманд git.

1/ добавляет содержимое файла в индекс.

2/ выполняет бинарный поиск ошибки поистории фиксаций

3/ выводит, создает и удаляет ветки.

4/ проверяет и переключает на ветку.

5/ клонирует репозиторий в новый каталог.

6/ записывает изменения в католог (фиксация)

7. показывает изменения между коммитами, рабочими деревьями и др.

8/ загружает объекты и ссылки из другого репозитория

9. выводит строки, соответствующие шаблону

10/ создает пустой репозиторий гит или переинициализирует существующий.

11. показывает журналы фиксации.

12/ объединяет две или больше истории разработки.

13. перемещает или переименовывает файл, каталог или символическую ссылку.

14/ получает изменения из удаленного репозитория и объединяет их с локальным репозиторием или локалной веткой.

15. загружает изменения из вашего локального реп. в удаленный.

16/ строит ровную линию фиксаций.

17. сбрасывает текущий HEAD в указанное состояние.

18/ удаляет файлы из рабочего дерева и из индекса.

19. показывает различные типы объектов.

20/ показывает состояние рабочего дерева.

21/ создает, выводит, удаляет или проверяет тег объекта, подписанного с GPG.

22/ выводит номер версии Git.
